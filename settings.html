Acerca de

GABBA es una app de código abierto. Su código fuente puede encontrarse en GitHub.

Amazon Web Services

Esta aplicación utiliza Amazon S3, el servicio de almacenamiento en la nube de Amazon Web Services (AWS), para guardar archivos multimedia de forma segura.

Apache Cordova

Esta aplicación fue desarrollada utilizando Apache Cordova.

Firebase

El sistema de notificaciones push está gestionado por Firebase y su complemento Cloud Messaging, servicio de Google.

Google Fonts

Las fuentes Inter y Oswald, diseñadas por sus respectivos autores, fueron proporcionadas por Google Fonts.


Los iconos de esta app fueron proporcionados por Google Fonts, a través de su colección de Material Symbols.

Aplicación desarrollada por Sebastián Fraga
